# 2025年5月训练与饮食计划

## 月度目标
- 完成最终赛前准备，达到最佳比赛状态
- 最大化肌肉定义和血管化
- 保持肌肉饱满度
- 体重目标：61-62kg
- 体脂目标：降至5-6%

## 阶段划分
本月分为三个关键阶段：
1. **赛前冲刺期**（5月1日-15日）：最后的减脂和肌肉定义阶段
2. **过渡期**（5月16日-21日）：为Peak Week做准备
3. **Peak Week**（5月17日-23日）：最终状态调整

## 赛前冲刺期（5月1日-15日）

### 训练分割
采用高频率训练，注重肌肉泵感和定义：
1. 周一：胸部 + 三头肌 + 有氧
2. 周二：背部 + 腹肌 + 有氧
3. 周三：肩部 + 二头肌 + 有氧
4. 周四：腿部 + 腹肌
5. 周五：上半身综合 + 有氧
6. 周六：下半身综合 + 腹肌 + 有氧
7. 周日：造型练习 + 低强度有氧

### 训练特点
- 高重复次数（15-20次）
- 短休息时间（30-60秒）
- 大量超级组和三联组
- 每次训练后增加15-20分钟造型练习

### 详细训练计划（赛前冲刺期）

#### 周一：胸部 + 三头肌 + 有氧
1. **三联组A**（4轮）:
   - 上斜哑铃卧推：4组，15-18次
   - 平板飞鸟：4组，15-18次
   - 绳索夹胸：4组，18-20次
2. **三联组B**（3轮）:
   - 窄距卧推：3组，15-18次
   - 绳索下压：3组，15-18次
   - 颈后臂屈伸：3组，15-18次
3. **造型练习**：15分钟（胸部和手臂展示姿势）
4. **有氧训练**：25分钟斜坡快走，心率130-140bpm

#### 周二：背部 + 腹肌 + 有氧
1. **三联组A**（4轮）:
   - 宽握引体向上：4组，尽可能多次数
   - 坐姿划船：4组，15-18次
   - 直臂下拉：4组，15-18次
2. **三联组B**（3轮）:
   - 单臂哑铃划船：3组，15-18次/侧
   - 反向飞鸟：3组，15-20次
   - 超人式挺身：3组，15-20次
3. **腹肌训练**:
   - 悬垂举腿：4组，15-20次
   - 卷腹：4组，20-25次
   - 侧卷腹：3组，20次/侧
4. **有氧训练**：30分钟椭圆机，中等强度

#### 周三：肩部 + 二头肌 + 有氧
1. **三联组A**（4轮）:
   - 坐姿哑铃肩推：4组，15-18次
   - 侧平举：4组，15-18次
   - 前平举：4组，15-18次
2. **三联组B**（3轮）:
   - 杠铃弯举：3组，15-18次
   - 交替哑铃弯举：3组，15-18次/臂
   - 集中弯举：3组，15-18次
3. **造型练习**：15分钟（肩部和手臂展示姿势）
4. **有氧训练**：20分钟HIIT（30秒冲刺/60秒恢复）

#### 周四：腿部 + 腹肌
1. **三联组A**（4轮）:
   - 杠铃深蹲：4组，15-18次
   - 箭步蹲：4组，15-18次/腿
   - 腿屈伸：4组，15-20次
2. **三联组B**（3轮）:
   - 坐姿腿弯举：3组，15-18次
   - 站姿小腿提升：3组，20-25次
   - 坐姿小腿提升：3组，20-25次
3. **腹肌训练**:
   - 仰卧交替触脚：4组，30次
   - 平板支撑：4组，60秒
   - 俄罗斯转体：3组，30次

#### 周五：上半身综合 + 有氧
1. **循环训练**（完成4轮，每个动作15-20次，动作间休息30秒，轮间休息2分钟）:
   - 哑铃卧推
   - 哑铃划船
   - 哑铃肩推
   - 哑铃侧平举
   - 哑铃弯举
   - 哑铃颈后臂屈伸
2. **造型练习**：20分钟（上半身所有展示姿势）
3. **有氧训练**：25分钟动感单车，中等强度

#### 周六：下半身综合 + 腹肌 + 有氧
1. **循环训练**（完成4轮，每个动作15-20次，动作间休息30秒，轮间休息2分钟）:
   - 哑铃深蹲
   - 保加利亚分腿蹲
   - 罗马尼亚硬拉
   - 单腿臀桥
   - 站姿小腿提升
   - 坐姿小腿提升
2. **腹肌训练**:
   - 悬垂举腿：4组，15-20次
   - AB轮：4组，12-15次
   - 侧平板支撑：3组，45秒/侧
3. **有氧训练**：30分钟快走，心率130-140bpm

#### 周日：造型练习 + 低强度有氧
1. **全面造型练习**：45-60分钟，包括：
   - 所有强制姿势练习
   - 过渡动作练习
   - 自选姿势练习
   - 四分之一转身练习
2. **低强度有氧**：45分钟步行，心率120-130bpm

## 赛前冲刺期饮食计划

### 热量与宏量营养素
- 总热量：约2300-2500卡路里/天
- 蛋白质：2.8-3.0g/kg体重（约170-185g/天）
- 碳水化合物：2.5-3.0g/kg体重（约150-185g/天）
- 脂肪：0.7-0.8g/kg体重（约40-50g/天）

### 碳水循环策略
实施更严格的碳水循环以最大化脂肪燃烧：
- 高碳日（训练大肌群日）：185-200g碳水
- 中碳日（训练小肌群日）：150-170g碳水
- 低碳日（休息日）：100-120g碳水
- 每周安排1次重填日：250-300g碳水（通常在周日）

### 示例一日饮食（中碳日）

#### 早餐 (6:00)
- 蛋白质煎蛋（5个蛋白）
- 燕麦30g
- 蓝莓30g
- 黑咖啡或绿茶

#### 上午加餐 (9:00)
- 蛋白粉1勺
- 杏仁10g
- 青苹果1个

#### 午餐 (12:00)
- 鸡胸肉150g
- 糙米或藜麦100g
- 混合蔬菜300g
- 橄榄油3ml

#### 训练前 (14:30)
- 蛋白粉1勺
- 香蕉1/2根或燕麦饼干1块

#### 训练后 (17:00)
- 蛋白粉1勺
- 葡萄糖或白米饭30g

- 白鱼或虾150g
- 红薯或糙米80g
- 混合蔬菜350g

#### 睡前 (21:30)
- 酪蛋白蛋白粉1勺

- 乳清蛋白：每天2-3次，每次25-30g
- 酪蛋白：睡前25-30g

- 多种维生素：每天1次

- 谷氨酰胺：训练后5g
- 维生素C：每天1000-2000mg（有助于减少皮下水分）

## 过渡期（5月16日-21日）

### 训练安排
- **5月16日（周五）**：背部训练，高碳日
- **5月17日（周六）**：腿部训练，开始水钠负荷
- **5月18日（周日）**：胸部训练，继续水钠负荷
- **5月19日（周一）**：背部训练，继续水钠负荷
- **5月20日（周二）**：手臂训练，开始减少水分摄入
- **5月21日（周三）**：全身轻量训练，开始碳水填充

### 训练特点
- 训练量逐渐减少（约为正常训练的70%）
- 重量保持不变，但减少组数
- 重点放在肌肉充血和泵感上
- 避免过度疲劳和肌肉损伤

### 饮食特点
- 开始水钠负荷（每天8-10L水）
- 保持高蛋白摄入
- 碳水逐渐调整，为Peak Week做准备
- 减少纤维摄入，避免胀气

## Peak Week（5月17日-23日）

### 详细计划

#### 5月17日（周六）：腿部训练日
- **训练**：腿部训练，中等强度，注重泵感
- **水分**：8L水
- **钠**：正常摄入，略高（约3-4g）
- **碳水**：1.5-2g/kg体重（约100-120g）
- **特别注意**：开始水钠负荷

#### 5月18日（周日）：胸部训练日
- **训练**：胸部训练，中等强度
- **水分**：8L以上
- **钠**：高钠（约4-5g）
- **碳水**：1g/kg体重（约60g）
- **特别注意**：继续水钠负荷

#### 5月19日（周一）：背部训练日
- **训练**：背部训练，中等强度
- **水分**：8L以上
- **钠**：高钠（约4-5g）
- **碳水**：0.5g/kg体重（约30g）
- **特别注意**：继续水钠负荷

#### 5月20日（周二）：手臂训练日
- **训练**：手臂训练，轻量级
- **水分**：8L以上
- **钠**：开始减少（约1-2g）
- **碳水**：0.5g/kg体重（约30g）
- **特别注意**：最后一天高水摄入

#### 5月21日（周三）：全身轻量训练日
- **训练**：全身轻量训练，注重泵感
- **水分**：6L
- **钠**：低钠（约0.5-1g）
- **碳水**：1.5g/kg体重（约90g），开始碳水填充
- **特别注意**：开始减少水分摄入

#### 5月22日（周四）：轻量训练或休息日
- **训练**：非常轻的泵感训练或完全休息
- **水分**：2-3L
- **钠**：零钠
- **碳水**：冲碳（4-5g/kg体重，约250-300g）
- **特别注意**：显著减少水分，增加碳水

#### 5月23日（周五）：完全休息日
- **训练**：只进行造型练习，不做其他训练
- **水分**：0.5-1L
- **钠**：零钠
- **碳水**：继续冲碳（4-5g/kg体重，约250-300g）
- **特别注意**：最小化活动量，保存能量

#### 5月24日（周六）：比赛日
- **早晨**：根据起床状态决定是否需要额外碳水
- **水分**：极少量（小口啜饮）
- **钠**：零钠
- **碳水**：根据身体状态调整（通常为少量快速碳水）
- **特别注意**：
  - 比赛前2-3小时停止进食固体食物
  - 比赛前适当热身以增加血管显现度
  - 按计划涂抹美黑剂和亮油

## 本月特别注意事项
1. 严格遵守Peak Week计划，不要临时改变策略
2. 密切监测身体状态，尤其是水分和碳水反应
3. 保持充分休息，避免不必要的体力消耗
4. 提前准备好所有比赛用品和食物
5. 保持积极心态，避免赛前焦虑
6. 记录整个过程的经验，为未来比赛提供参考

## 赛后恢复计划
- 比赛结束后逐渐恢复正常饮食，避免暴饮暴食
- 第一周以中等热量饮食为主，逐渐增加碳水
- 比赛后1-2周进行轻量恢复性训练
- 比赛后进行全面评估，总结经验教训
