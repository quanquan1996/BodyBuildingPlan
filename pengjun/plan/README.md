# 彭军健美训练计划

本文件夹包含彭军从2025年3月至5月的健美训练和饮食计划，为2025年5月24日的比赛做准备。

## 计划概览

- [3月计划](./March2025.md) - 新赛季开始，重点增肌和力量训练
- [4月计划](./April2025.md) - 过渡期，开始减脂同时保持肌肉量
- [5月计划](./May2025.md) - 赛前最后冲刺，专注于肌肉定义和比赛准备

## 基本信息

- 身高：1.67m
- 体重：62.5kg
- 体脂率：8%
- 训练年限：6年

## 总体目标

1. 保持或略微增加肌肉量
2. 降低体脂至比赛水平（约5-6%）
3. 提高肌肉质量和定义
4. 优化身体比例
5. 完善比赛展示技巧
