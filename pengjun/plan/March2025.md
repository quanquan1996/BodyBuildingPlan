# 2025年3月训练与饮食计划

## 月度目标
- 开始新赛季的增肌期
- 提高整体力量水平
- 建立良好的训练习惯和饮食规律
- 体重目标：63-64kg
- 体脂目标：维持在8-9%

## 训练计划

### 训练分割
采用5天分化训练，2天休息：
1. 周一：胸部 + 三头肌
2. 周二：背部 + 肱二头肌
3. 周三：休息
4. 周四：肩部 + 腹肌
5. 周五：腿部
6. 周六：手臂 + 弱项加强
7. 周日：休息

### 训练强度与容量
- 每个大肌群组8-10个动作，3-4组
- 每个小肌群组4-6个动作，3组
- 组间休息：大肌群2-3分钟，小肌群1-2分钟
- 重量范围：8-12次重复为主，部分基础动作可采用5-8次的重量训练
- 训练时长：60-75分钟/次

### 详细训练计划

#### 周一：胸部 + 三头肌
1. 平板杠铃卧推：4组，8-10次
2. 上斜哑铃卧推：4组，10-12次
3. 下斜哑铃飞鸟：3组，12-15次
4. 坐姿器械夹胸：3组，12-15次
5. 绳索下压：3组，12-15次
6. 窄距卧推：3组，10-12次
7. 哑铃颈后臂屈伸：3组，12-15次

#### 周二：背部 + 肱二头肌
1. 引体向上：4组，尽可能多次数
2. 杠铃划船：4组，8-10次
3. 单臂哑铃划船：3组，10-12次/侧
4. 高位下拉：3组，10-12次
5. 直臂下拉：3组，12-15次
6. 杠铃弯举：3组，10-12次
7. 交替哑铃弯举：3组，12-15次

#### 周四：肩部 + 腹肌
1. 坐姿哑铃肩推：4组，8-10次
2. 侧平举：4组，12-15次
3. 俯身侧平举：3组，12-15次
4. 绳索面拉：3组，15-20次
5. 杠铃直立划船：3组，10-12次
6. 悬垂举腿：4组，15-20次
7. 卷腹：4组，20-25次
8. 侧卷腹：3组，20次/侧

#### 周五：腿部
1. 杠铃深蹲：5组，8-10次
2. 腿举：4组，10-12次
3. 保加利亚分腿蹲：3组，10-12次/腿
4. 腿屈伸：3组，12-15次
5. 坐姿腿弯举：4组，12-15次
6. 站姿小腿提升：4组，15-20次
7. 坐姿小腿提升：3组，15-20次

#### 周六：手臂 + 弱项加强
1. 杠铃弯举：4组，8-10次
2. 锤式弯举：3组，10-12次
3. 集中弯举：3组，12-15次
4. 窄距卧推：4组，8-10次
5. 绳索下压：3组，12-15次
6. 杠铃颈后臂屈伸：3组，10-12次
7. 弱项训练：根据个人情况选择2-3个动作，各3组

## 有氧训练
- 每周3-4次，安排在力量训练后或休息日
- 时长：20-30分钟
- 强度：中低强度（心率在120-140bpm）
- 推荐形式：快走、慢跑、椭圆机、动感单车

## 饮食计划

### 热量与宏量营养素
- 总热量：约2800-3000卡路里/天
- 蛋白质：2.2-2.5g/kg体重（约140-160g/天）
- 碳水化合物：4-5g/kg体重（约250-320g/天）
- 脂肪：1g/kg体重（约60-65g/天）

### 餐次安排
- 每天5-6餐，包括3次主餐和2-3次加餐
- 训练前1-2小时进食含碳水和蛋白质的餐点
- 训练后30分钟内补充蛋白质和碳水化合物

### 示例一日饮食

#### 早餐 (7:00)
- 全麦面包2片
- 全蛋2个+蛋白3个
- 燕麦50g
- 蓝莓或其他浆果一小把
- 黑咖啡或绿茶

#### 上午加餐 (10:00)
- 希腊酸奶200g
- 混合坚果30g
- 香蕉1根

#### 午餐 (13:00)
- 鸡胸肉150g
- 糙米/红薯/土豆200g
- 混合蔬菜200g
- 橄榄油10ml

#### 训练前 (15:30)
- 香蕉1根
- 蛋白粉1勺(约25g蛋白质)
- 燕麦饼干2块

#### 训练后 (18:00)
- 蛋白粉1勺
- 快速碳水(葡萄糖或白米饭)50g

#### 晚餐 (20:00)
- 瘦牛肉/鱼/虾150g
- 糙米/红薯/藜麦150g
- 混合蔬菜250g
- 橄榄油或亚麻籽油10ml

#### 睡前 (22:30，可选)
- 酪蛋白蛋白粉1勺
- 杏仁奶200ml

### 补充剂建议
- 乳清蛋白：每天1-2次，每次25-30g
- 肌酸：每天5g
- 多种维生素：每天1次
- 鱼油：每天2-3g
- 维生素D：每天2000-5000IU（根据血检结果调整）
- ZMA：睡前服用（可选）

## 恢复策略
- 保证每晚7-8小时高质量睡眠
- 每周安排2次全身拉伸或瑜伽课
- 每周1-2次泡沫轴放松
- 每两周安排一次按摩（可选）
- 保持充分水分摄入，每天至少3-4升水

## 进度追踪
- 每周一次体重测量（早晨空腹）
- 每两周一次体围测量（胸、腰、臀、大腿、上臂）
- 每月一次体脂测量
- 训练日志记录（重量、组数、次数）
- 每周拍照记录身体变化

## 调整机制
- 如体重两周内未增加，可增加每日200-300卡路里
- 如体脂明显增加，可减少碳水摄入或增加有氧训练频率
- 根据恢复情况调整训练量和强度
- 根据能量水平和训练表现调整碳水摄入时机和数量

## 本月特别注意事项
1. 重点关注基础复合动作的技术和进步
2. 建立稳定的饮食和训练习惯
3. 确保充分的恢复和睡眠
4. 记录训练和饮食日志，为后续月份提供参考
5. 避免过度训练，专注于质量而非数量
