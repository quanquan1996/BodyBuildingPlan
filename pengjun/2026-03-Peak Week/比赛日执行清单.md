# 比赛日执行清单

## 比赛日时间安排

### 早晨起床后（比赛前4-5小时）
- [ ] 测量体重并记录
- [ ] 评估整体状态（饱满度、干度）
- [ ] 拍照记录状态
- [ ] 根据状态决定碳水摄入量

### 状态评估标准
**理想状态：** 饱满+干燥，直接维持
**偏干：** 适量碳水补充（50-100g）
**偏水：** 减少或停止碳水摄入

### 比赛前2-3小时
- [ ] 最后一次少量碳水补充（如需要）
- [ ] 停止饮水（仅润唇）
- [ ] 开始准备比赛用品

### 比赛前1小时
- [ ] 到达比赛场地
- [ ] 轻度热身和充血
- [ ] 涂抹比赛油彩
- [ ] 最后状态检查

## 充血训练方案
**目标：** 增加肌肉血流，提升视觉效果
**时间：** 比赛前30-45分钟

### 上半身充血
- 俯卧撑：2组 x 15-20次
- 哑铃弯举：2组 x 12-15次
- 侧平举：2组 x 12-15次
- 三头屈伸：2组 x 12-15次

### 下半身充血
- 深蹲：2组 x 15-20次
- 弓步蹲：2组 x 10-12次/腿
- 提踵：2组 x 15-20次

**注意：** 轻重量，高次数，避免疲劳

## 比赛用品检查清单
- [ ] 比赛服装
- [ ] 比赛油彩（底色+亮油）
- [ ] 毛巾
- [ ] 应急碳水（香蕉、葡萄糖）
- [ ] 少量饮用水
- [ ] 充血用小哑铃
- [ ] 镜子（状态检查）

## 心理准备
- [ ] 回顾成功经验，建立信心
- [ ] 放松心态，享受比赛过程
- [ ] 专注于展示最佳状态
- [ ] 相信自己的准备和能力

## 比赛中注意事项
1. **造型展示**：自信、稳定、突出优势部位
2. **呼吸控制**：避免憋气过久
3. **状态维持**：比赛间隙适当充血
4. **心态调整**：专注当下，发挥最佳

## 比赛后

- [ ] 拍照留念
- [ ] 开始恢复饮食
- [ ] 总结整个备赛过程
