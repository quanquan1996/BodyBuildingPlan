# 2026年3月健美比赛备赛计划设计文档

## 概述

本设计文档详细描述了为彭军制定的从2025年7月到2026年3月的系统化健美比赛备赛计划。基于用户当前状况（66kg，15%体脂）和丰富比赛经验，采用三阶段策略：增肌期（5个月）→ 减脂期（2个月）→ Peak Week（1周）。

## 架构设计

### 整体时间线架构

```
2025年7月-12月：增肌期（5个月）
├── 目标：66kg → 69-71kg
├── 体脂控制：15% → 16-18%
└── 重点：力量提升 + 肌肉量增加

2026年1月-2月：减脂期（2个月）
├── 目标：69-71kg → 59-61kg
├── 体脂控制：16-18% → 5-6%
└── 重点：保持肌肉 + 精准减脂

2026年3月：Peak Week + 比赛
├── 目标：达到最佳竞技状态
├── 基于2025年5月成功经验
└── 重点：水钠调控 + 碳水填充
```

## 组件和接口设计

### 1. 增肌期训练组件

**训练分化方案：**
- 推拉腿分化（每周2轮，共6天训练）
- 每个肌群每周训练频率：2次
- 训练强度：75-85% 1RM为主
- 组数：每个肌群每周16-20组

**渐进超负荷策略：**
- 第1-2周：适应期，建立动作模式
- 第3-4周：重量递增5-10%
- 每月评估并调整训练重量

### 2. 营养计算组件

**增肌期营养参数：**
- 基础代谢率（BMR）：约1720卡路里
- 总消耗（TDEE）：约2400卡路里
- 目标摄入：2700-2900卡路里（盈余300-500）
- 宏量营养素比例：
  - 蛋白质：2.2g/kg体重（约145g）
  - 脂肪：1g/kg体重（约66g）
  - 碳水化合物：剩余热量（约370-420g）

**减脂期营养参数：**
- 目标摄入：1800-2000卡路里（赤字300-500）
- 宏量营养素比例：
  - 蛋白质：2.5g/kg体重（保护肌肉）
  - 脂肪：0.8g/kg体重
  - 碳水化合物：根据训练量调整

### 3. Peak Week调控组件

**基于2025年5月成功经验的方案：**

**水钠负荷期（赛前7-5天）：**
- 饮水量：8-10L/天
- 钠摄入：正常或略高
- 碳水：维持低水平

**脱水期（赛前4-2天）：**
- 饮水递减：6L → 4L → 2L
- 严格零钠执行
- 碳水填充开始

**主充碳日（赛前2天）：**
- 碳水目标：600-650g（基于成功经验）
- 食物选择：白米饭、去皮土豆、鸡蛋白
- 监测指标：体重、分离度、饱满度

**调整日（赛前1天）：**
- 碳水：200-300g（保守策略）
- 饮水：极少量
- 状态微调

### 4. 监测评估组件

**每周监测指标：**
- 体重（每日晨起空腹）
- 体脂率（每周同一时间）
- 关键围度（胸围、臂围、腰围、腿围）
- 力量指标（主要动作1RM测试）

**照片记录标准：**
- 固定时间：每周日晨起空腹
- 固定光线：自然光或标准灯光
- 固定角度：正面、侧面、背面
- 固定姿势：放松状态 + 紧张状态

## 错误处理

### 增肌期异常处理
- **体重增长过快**：减少热量摄入100-200卡路里
- **力量停滞**：调整训练量或安排减量周
- **消化不良**：调整食物选择，增加消化酶

### 减脂期异常处理
- **减重过快**：增加热量摄入，减少有氧
- **肌肉流失**：提高蛋白质摄入，调整训练
- **平台期**：重新计算TDEE，调整营养方案

### Peak Week异常处理
- **碳水溢出**：立即减少碳水摄入，增加轻度活动
- **脱水不足**：检查隐藏钠源，调整饮水策略
- **状态平坦**：适量补充碳水，轻度充血训练

## 测试策略

### 阶段性测试点
1. **月度评估**：体重、体脂、围度、力量测试
2. **阶段转换测试**：增肌期结束时全面身体评估
3. **Peak Week预演**：赛前4周进行小规模Peak Week测试

### 成功指标
- **增肌期**：体重增加2.5-4.5kg，力量提升15-25%
- **减脂期**：保持90%以上肌肉量，达到目标体脂
- **Peak Week**：达到最佳干度和饱满度平衡

### 风险评估
- **伤病风险**：渐进式训练，充分热身
- **营养风险**：定期血液检查，补充必需营养素
- **心理风险**：设置合理期望，保持积极心态

## 补剂策略

### 基础补剂（全程使用）
- 乳清蛋白：训练后30g
- 肌酸：每日5g
- 复合维生素：每日1粒
- 鱼油：每日2-3g

### 增肌期特殊补剂
- 增重粉：训练日使用
- 消化酶：随餐服用
- ZMA：睡前服用

### 减脂期特殊补剂
- 左旋肉碱：训练前2g
- 绿茶提取物：餐前服用
- CLA：随餐服用

### Peak Week补剂
- 基于成功经验：Q10、维生素C
- 应急补剂：西甲硅油（胀气）、开塞露（便秘）
